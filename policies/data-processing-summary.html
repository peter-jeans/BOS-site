Data Processing Summary

Last updated: 18 June 2026.

Local BOS

Local BOS stores project-specific memory, decisions, evidence records, preferences, and development context in project-controlled files.

Base44 direct BOS

Base44 apps should receive compact guidance, onboarding status, policy pointers, and Cloud BOS connection instructions. They should not receive private BOS internals or full local project memory.

Cloud BOS

Cloud BOS should provide compact guidance, upgrade metadata, runtime checks, agreement status, entitlement status, and sanitized learning records.

Excluded by default

  • Raw source code, secrets, keys, credentials, user content, payment data, private customer data, database exports, raw logs, raw evidence records, full prompts, and full private project memory.

Future capsules

Any richer project capsule, evidence bundle, export, or Cloud sync feature should require explicit consent, clear data classification, retention controls, and deletion/export rights.
