Data Processing Summary

Last updated: 18 June 2026.

Local BOS

Local BOS stores project-specific governance memory, case law, receipts, preferences, and development context in local project files such as `.bos` and DevPack records.

Base44 direct BOS

Base44 apps should receive compact AICONTROL guidance, onboarding status, policy pointers, and Cloud BOS connection instructions. They should not receive full masterBOS internals or full local project memory.

Cloud BOS

Cloud BOS should provide compact manifests, advisory rules, upgrade metadata, runtime checks, agreement status, entitlement status, and sanitized learning records.

Excluded by default

  • Raw source code, secrets, keys, credentials, user content, payment data, private customer data, database exports, raw logs, raw receipts, full prompts, full `.bos`, and full DevPack material.

Future capsules

Any richer project capsule, evidence bundle, export, or Cloud sync feature should require explicit consent, clear data classification, retention controls, and deletion/export rights.
